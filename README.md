## 项目起源

> **EE-Wiki** 致力于成为一个免费开放且持续更新的电子行业相关知识整合站点，从模电数电基础知识，到FPGA开发、嵌入式开发,再到电赛、智能车赛等...我们希望能够包涵电子行业所需所有知识，成为一个知识宝库。

*本项目受 [OI Wiki](https://oi-wiki.org/) 的启发，在编写过程中参考了许多，在此表示敬意。*
