## 简介

&emsp;&emsp;VerilogHDL是1983年由GDA(Gateway Design Automation)公司的Phil Moorby首创，之后Moorby又设计了Verilog-XL仿真器，并大获成功，也使得VerilogHDL语言得到推广使用。[^1]
